PROJECT TITLE: Soal 2 Tugas GUI

PURPOSE OF PROJECT:
class LayoutTester yang memiliki 5 komponen JButton didalamnya. Class LayoutTester dapat 
mengubah layout dari kelima JButton. Jika method setFlowLayout() yang dipanggil, maka tampilan 
kelima-lima JButton menggunakan layout FlowLayout. Jika method setBorderLayout() yang dipanggil, 
maka tampilan kelima-lima JButton menggunakan layout BorderLayout. Jika method setGridLayout() 
yang dipanggil, maka tampilan kelima-lima JButton menggunakan layout GridLayout.

VERSION or DATE: 23/11/2022

AUTHORS: Marlina

USER INSTRUCTIONS:
Pada bagian main
1. Untuk menampilkan kelima JButton menggunakan layout BorderLayout 
   maka pada bagian l1.setBorderLayout(); untuk memanggil setBorder.
2. Untuk menampilkan kelima JButton menggunakan layout FlowLayout 
   maka pada bagian l1.setFlowLayout(); untuk memanggil setFlow.
3. Untuk menampilkan kelima JButton menggunakan layout BorderLayout 
   maka pada bagian l1.setGridLayout(); untuk memanggil setGrid.
